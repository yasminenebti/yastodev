# Bracket Combinations

## 🧩 Problem

Have the function BracketCombinations(num) take the num parameter being passed which will be an integer greater than or equal to zero and return the number of valid combinations that can be formed with num pairs of parentheses. For example, if the input is 3, then the possible combinations of 3 pairs of parenthesis, namely: ()()(), ()(()), (())(), ((())), and (()()) are all valid and for 3, there are 5 valid combinations.

## 📌 Examples

Input: 3  
Output: 5

Input: 2  
Output: 2

Input: 1  
Output: 1
