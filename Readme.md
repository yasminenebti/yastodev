# Yastodev

Yastodev is a personal learning path by Yasmine Nebti, a software developer passionate about building her skills and becoming an excellent software engineer. This repository is dedicated to tracking progress, experimenting with code, and consolidating knowledge in software development and devops.

---

## **Getting Started**

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/yasminenebti/yastodev.git
   cd yastodev
   ```

2. **Build the Project**:
   Ensure you have Maven installed, then build the project:
   ```bash
   mvn clean install
   ```

3. **Run Individual Modules**:
   Navigate to the desired module and run the code using your IDE or Maven.
